# Meeting Prep Prompts

Show up prepared. Win the room. Close the deal.

---

## Pre-Meeting Research

### Quick Prep (5 Minutes)

```
I have a meeting with [PERSON], [TITLE] at [COMPANY] in 5 minutes.

Quick hits:
1. What do they do (one sentence)
2. Why they might care about [MY PRODUCT]
3. One personalization hook (their background or recent news)
4. Opening question to ask
5. One risk to avoid
```

---

### Full Prep Brief (15 Minutes)

```
Create a meeting prep brief for my call with [PERSON], [TITLE] at [COMPANY]:

My product: [WHAT YOU SELL]
Meeting purpose: [DISCOVERY / DEMO / NEGOTIATION / etc.]
Time allocated: [MINUTES]

Include:

1. PERSON PROFILE
- Their role and likely priorities
- Background and career history
- Recent LinkedIn activity or content
- Communication style clues

2. COMPANY CONTEXT
- What they do
- Recent news (funding, launches, hires)
- Likely challenges
- Competitors they face

3. MEETING STRATEGY
- Opening approach
- Key questions to ask
- Points to make
- What NOT to do

4. EXPECTED OBJECTIONS
- What they might push back on
- How to handle each

5. DESIRED OUTCOME
- What success looks like
- Specific ask at the end
- Backup ask if needed
```

### Stakeholder Map

```
I have a meeting with multiple stakeholders at [COMPANY]:

Attendees:
1. [NAME, TITLE]
2. [NAME, TITLE]
3. [NAME, TITLE]

Help me understand:
1. Each person's likely agenda
2. Who's the decision maker
3. Who might be a blocker
4. How to engage each person
5. Questions to ask each role
```

## Meeting Agendas

### Discovery Call Agenda

```
Create an agenda for a [30/60]-minute discovery call with [PERSONA]:

Meeting goal: [WHAT YOU NEED TO LEARN]
My product: [WHAT YOU SELL]

Include:
1. Opening and rapport (X min)
2. Context setting (X min)
3. Discovery questions (X min)
4. Brief intro to solution (X min)
5. Next steps (X min)

With specific talking points for each section.
```

---

### Demo Agenda

```
Create an agenda for a demo with [COMPANY]:

Attendees: [LIST NAMES/TITLES]
Time: [MINUTES]
Their priorities: [WHAT THEY CARE ABOUT]
Features to show: [LIST]

Structure:
1. Re-confirm goals
2. Demo flow (feature by feature with transitions)
3. Q&A
4. Next steps

Include questions to ask during the demo.
```

---

### Negotiation Call Agenda

```
Create an agenda for a negotiation call with [COMPANY]:

Context:
- Where we are: [STAGE]
- Their concerns: [OBJECTIONS/ISSUES]
- What's on the table: [PRICING, TERMS]
- Our must-haves: [NON-NEGOTIABLES]
- Our flex points: [WHERE WE CAN MOVE]

Include:
1. Opening positioning
2. Key points to make
3. Questions to ask
4. Concession strategy
5. Closing approach
```

---

### QBR/Check-In Agenda

```
Create an agenda for a QBR/check-in with [CUSTOMER]:

Relationship context:
- How long they've been a customer: [TIME]
- Current usage: [HOW THEY USE YOU]
- Open issues: [ANY PROBLEMS]
- Expansion opportunity: [IF RELEVANT]

Structure:
1. Celebrate wins
2. Review metrics
3. Address concerns
4. Future roadmap
5. Expansion conversation (if appropriate)
6. Feedback
```

## Opening & Closing

### Meeting Openers

```
Give me 5 ways to open a [MEETING TYPE] with [PERSONA]:

Context: [WHAT THIS MEETING IS ABOUT]
Relationship: [FIRST MEETING / FOLLOW-UP / LONG-TERM]

Requirements:
- Build rapport without wasting time
- Set the right tone
- Transition smoothly to substance
- Match their communication style
```

---

### Meeting Closers

```
Give me language to close a [MEETING TYPE] and get commitment to next steps:

Meeting was about: [TOPIC]
What I want next: [SPECIFIC NEXT STEP]
Their likely hesitation: [WHAT MIGHT SLOW THEM DOWN]

Provide:
1. Transition to close
2. Summary of key points
3. Ask for next step
4. Alternative if they hesitate
5. Confirm timeline
```

## During the Meeting

### Real-Time Pivot

```
I'm in a meeting and things aren't going as planned:

What's happening: [DESCRIBE SITUATION]
My original goal: [WHAT I WANTED]
Their signals: [WHAT THEY'RE SAYING/DOING]

Help me:
1. Read the situation
2. Pivot strategy
3. Salvage the meeting
4. Adjust my ask
5. Graceful exit if needed
```

---

### Handle Unexpected Attendees

```
Unexpected people joined my meeting:

Original attendees: [WHO I EXPECTED]
New attendees: [WHO SHOWED UP]
Their likely role: [MY GUESS]
My prepared content: [WHAT I WAS GOING TO COVER]

How do I:
1. Adjust my approach
2. Re-discover their priorities
3. Handle multiple agendas
4. Still achieve my goal
```

## Post-Meeting

### Meeting Summary Email

```
Write a follow-up email summarizing my meeting with [PERSON/TEAM]:

Meeting notes:
[PASTE YOUR NOTES]

Include:
1. Thanks (brief)
2. Key points discussed
3. Agreed next steps (with owners and dates)
4. Anything I promised to send
5. Clear ask
```

---

### Internal Debrief

```
Create an internal debrief from this meeting:

[PASTE NOTES OR TRANSCRIPT]

Format for Slack/team update:
1. TL;DR (one sentence)
2. Key takeaways (bullets)
3. Decision makers and dynamics
4. Concerns/objections raised
5. Next steps
6. Risk assessment
7. What I need help with
```

---

### CRM Notes

```
Format these meeting notes for my CRM:

[PASTE RAW NOTES]

Format as:
1. Meeting date and attendees
2. Key pain points discussed
3. Decision process and timeline
4. Budget indicators
5. Competition mentioned
6. Next steps (with dates)
7. Deal stage recommendation
8. Follow-up tasks
```

## Specific Meeting Types

### First Meeting with Executive

```
Prep me for a first meeting with [C-LEVEL TITLE] at [COMPANY]:

What I'm selling: [PRODUCT]
Why they took the meeting: [TRIGGER/CONTEXT]
Time allocated: [MINUTES]

Executives expect:
1. Get to the point
2. Strategic conversation
3. Bottom-line impact
4. Don't waste time on features

Create:
1. Opening (30 seconds)
2. 3 strategic questions to ask
3. How to position my solution (exec-level)
4. Clear ask
5. What NOT to do
```

---

### Technical Deep-Dive

```
Prep me for a technical deep-dive with [TECHNICAL PERSONA] at [COMPANY]:

My product: [WHAT YOU SELL]
Technical aspects: [KEY TECHNICAL FEATURES]
Their likely concerns: [SECURITY, INTEGRATION, etc.]

Create:
1. Technical topics to cover
2. Questions they'll ask (and answers)
3. Demo flow for technical audience
4. How to handle "I'll need to test it"
5. Technical proof points to share
```

---

### Procurement/Legal Call

```
Prep me for a procurement/legal call at [COMPANY]:

What we've agreed on: [DEAL TERMS]
Contract status: [WHERE WE ARE]
Known concerns: [WHAT THEY'VE RAISED]
My flexibility: [WHERE I CAN MOVE]

Create:
1. Expected asks from their side
2. How to respond to each
3. What to push back on
4. How to keep momentum
5. Escalation path if stuck
```

---

### Multi-Stakeholder Presentation

```
Prep me for presenting to multiple stakeholders at [COMPANY]:

Attendees:
- [NAME, TITLE, THEIR AGENDA]
- [NAME, TITLE, THEIR AGENDA]
- [NAME, TITLE, THEIR AGENDA]

Time: [MINUTES]
Goal: [WHAT YOU NEED TO ACHIEVE]

Create:
1. Opening that acknowledges all agendas
2. Content that speaks to each persona
3. Questions to ask to each person
4. How to manage competing priorities
5. Closing that gets consensus
```
