Why they stall — it's never the price

🔑 Primary driver of ghosting
Loss aversion. Boutique founders fear the ego cost of a failed adoption more than they value the gain. Every ghost is identity protection, not budget protection. They've already paid for 2–4 tools — admitting those don't work means admitting those choices were wrong.
📊 Behavioral science finding
2.3x more likely to act when their current situation is reframed as risky vs. when future benefits are emphasized. Lead with cost of inaction, not product features. "Here's what you're losing every day" beats "imagine what's possible."
🪪 Identity exposure
Owner = firm. Every tool decision is personal. These operators sell their own judgment to clients — admitting their stack is broken contradicts their professional self-image. Make adoption the sophisticated move, not the risky one.

Use: "You already know your tools aren't working together. That's not a character flaw — it's a structural problem nobody's solved."
⏱️ Time is the ROI currency
Willingness to pay is tied to hours recovered, not feature count. Platforms with 40+ features consistently lose to those with 15 clearly presented ones. Cognitive overload = soft no. Keep onboarding under 20 minutes or loss aversion activates.
💡 Inaction cost frame (use post-call, not during)
Operators running 3 AI tools spend 5–15 hours/week as the manual relay. At $200/hr billing rate: $1,000–$3,000/month in invisible overhead. COMMAND pilot: $99/month. ROI payback: first week.
🗣️ Always say "pilot" not "trial"
These operators use "pilot" with their own clients. It signals peer-level sophistication, lowers identity exposure, and frames the test as professional — not consumer software. This is a hard language rule across all copy.

Technical terms in plain English + military narrative guide

⚙️ MCP — Model Context Protocol
Open protocol created by Anthropic in 2024. Lets AI agents report their state, capabilities, and outputs to a central host in a standardized way. COMMAND becomes the MCP host — the thing that receives and coordinates those reports. Anthropic made the protocol but doesn't own the host layer. We claim it. Once an operator registers agents through COMMAND as their MCP host, Anthropic's native features become additive, not competitive.
⚙️ A2A — Agent-to-Agent Protocol
Standardized protocol (finalized 2025–2026) for agents to communicate with each other across different platforms. MCP + A2A together are the integration layer COMMAND sits above. These protocols are why now is the right time — the coordination problem couldn't be solved at scale before they existed.
⚙️ CCF — Context Checkpoint Format
COMMAND's proprietary format for packaging and transferring context between agents. When one agent hands off work to another, the CCF ensures nothing is lost. Stored as JSONB in Supabase. Part of the MACP patent claim. Trade secret — private repo + NDA.
⚙️ MACP — Multi-Agent Coordination Protocol
AEP (Agent Event Protocol) + CCF + Task Router routing schema combined into a unified cross-vendor coordination protocol. Primary patentable asset. Provisional spec complete. Gate: first paying customer. Jason qualifies as micro entity — 75% USPTO fee discount.
⚙️ BYOA — Bring Your Own Agents
COMMAND does not provide AI models. Customers bring their existing Claude, GPT, Perplexity, Cursor subscriptions. COMMAND coordinates them. Model-agnostic by architecture. No lock-in to any single vendor.
⚙️ Stack (say this simply)
Next.js 15 / React 19 — modern web framework. Supabase — database + auth + real-time (AES-256 encryption confirmed). Vercel — deployment (same platform as Anthropic, Linear). Stripe — payments.
