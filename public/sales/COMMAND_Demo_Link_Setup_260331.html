Setup — one-time, ~15 minutes

1
Create Dub.co account Go to dub.co — free tier. Sign up with jason@globalinkservices.io. No credit card required at free tier.
2
Add your domain Dub.co → Settings → Domains → Add Domain → enter globalinkservices.io. Dub shows you a CNAME record to add.
3
Add CNAME in DNS (Vercel) Vercel Dashboard → your globalinkservices.io project → Settings → Domains → Add.
Type: CNAME Name: go Value: cname.dub.co Repeat for subdomain: demo → cname.dub.co Repeat for subdomain: trial → cname.dub.co
Propagation: 5–15 min. Dub.co will confirm when active.
4
Create link 1 — one-pager
Dub.co → Create Link Short link: go.command.globalinkservices.io Destination: https://command.globalinkservices.io/one-pager Tags: sales, one-pager
5
Create link 2 — demo
Dub.co → Create Link Short link: demo.command.globalinkservices.io Destination: [paste Navattic URL after Part 2 below] Tags: sales, demo
Leave destination blank until Navattic is published. Update it after.
6
Create link 3 — trial
Dub.co → Create Link Short link: trial.command.globalinkservices.io Destination: https://app.command.globalinkservices.io/signup Tags: sales, trial
7
Test all three links Open each in a private browser window. Confirm redirect lands on correct destination. Check Dub.co dashboard — clicks should show immediately.
