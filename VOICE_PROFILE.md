# JASON DAVIS — VOICE PROFILE
Last updated: April 6, 2026
This file is the canonical voice reference for all COMMAND outreach copy.
Update it every time Jason shares actual sent messages.

## Opener
Format: `[First Name] —` (em-dash after name, nothing else on that line)
New paragraph starts immediately after. No trailing text on the opener line.

## Structure
- Short blocks — 2-4 sentences max per paragraph
- A single sentence can stand alone as its own paragraph for rhythm
- Two line breaks between paragraphs — mandatory for copy-paste fidelity
- Plain language. Write like you talk.

## Tone
Peer to peer. Direct and warm. Curious, not pitching.
Warmth is real or absent — never performed.
Never salesy, never consultant-speak, never LinkedIn influencer voice.

## Banned Words and Phrases
- "I hope this finds you well"
- "I wanted to reach out" / "I wanted to follow up"
- "circle back" / "touch base" / "leverage" / "synergy"
- "in the trenches" / "meaningful work" / "solo operator" / "ICP"
- "pain points" / "value prop"
- "Would it be useful" / "Would that make sense"
- Anything that sounds like it came from a sales template

## Sign-Off
`JCMD` only. No dash before it. No punctuation after it.
Never: Jason / Best / Regards / Thanks / Warmly / Cheers

## CTA
Preferred: `Open for a call this week or next?`
Never: "Would a 20-minute call be useful?" or "Would you be open to..."

## Mutual Rule
If Jason offered help during a call or relationship, that appears in the follow-up.
Never send a message that only asks. Two-way always.
