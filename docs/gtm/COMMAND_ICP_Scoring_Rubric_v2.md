# Single-Tool Operator — Qualification Note

A prospect who runs ONLY Claude but operates 4+ simultaneous tabs/sessions is a TIER 1 candidate. The coordination problem is identical regardless of vendor diversity. Do not disqualify single-tool operators.

Follow-up question for single-tool prospects: "How many Claude sessions are you running simultaneously right now? And how do those sessions share context with each other?" The answer surfaces the core pain regardless of tool diversity.
